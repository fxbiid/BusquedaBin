# Búsqueda Binaria – Estructuras de Datos

**Autor:** Fabián Nicolás Díaz Navarrete  
**Fecha:** Enero 2026  

Implementación del algoritmo de **Búsqueda Binaria** en C++, utilizando un arreglo ordenado (`vector<int>`) para encontrar un elemento de forma eficiente.

El programa permite:

- Buscar un número en un arreglo ordenado.
- Retornar la posición si el dato existe.
- Informar cuando el número no se encuentra.

---

## Compilación

```bash
g++ -std=c++17 -O2 -Wall -Wextra -pedantic main.cpp -o BusquedaBinaria
./BusquedaBinaria
